**LUMINA** is a premium, high-impact landing page designed for an exclusive streaming space dedicated to raw cinematic masterpieces and groundbreaking originals. It features immersive animations, smooth scrolling, and a striking dark-mode aesthetic to provide a top-tier user experience.

## ✨ Features

- **Immersive Design:** A highly polished, dark-themed UI that evokes a cinematic feel.
- **Smooth Scrolling:** Powered by [Lenis](https://github.com/studio-freight/lenis) for a fluid, natural scroll experience.
- **Dynamic Animations:** Integration with [GSAP (GreenSock)](https://gsap.com/) and ScrollTrigger for sophisticated, scroll-driven visual effects.
- **Responsive Layout:** Perfectly adapts to mobile, tablet, and desktop viewports using modern CSS techniques.
- **Modern Stack:** Built on the blazingly fast [Astro](https://astro.build/) framework and styled with [Tailwind CSS v4](https://tailwindcss.com/).

## 🛠️ Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS (v4)](https://tailwindcss.com/)
- **Animations:** [GSAP](https://gsap.com/)
- **Smooth Scroll:** [Lenis](https://github.com/studio-freight/lenis)

## 🚀 Getting Started

Follow these instructions to set up the project locally.

### Prerequisites

- Node.js (v22.12.0 or higher recommended)
- `npm` or your preferred package manager (yarn, pnpm, bun)

### Installation

1. **Navigate to the project directory:**
   ```bash
   cd motion-one-landing
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:4321` to see the application in action.

## 🏗️ Build for Production

To create an optimized production build, run:

```bash
npm run build
```

You can then preview your production build locally with:

```bash
npm run preview
```

## 👨‍💻 Project Structure

- `src/components/`: Reusable Astro components (Hero, Marquee, Footer, etc.)
- `src/layouts/`: Base layout definitions
- `src/pages/`: Astro pages (e.g., `index.astro`)
- `src/styles/`: Global CSS and Tailwind properties
- `public/`: Static assets like images and favicons
